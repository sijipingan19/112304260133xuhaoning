### 1.6 分析问题（请逐条回答）

**Q1：Adam 和 SGD 的收敛速度有何差异？从实验结果中你观察到了什么？**

答：Adam优化器收敛速度明显快于SGD。在相同条件下，Adam在第10个epoch左右就能达到97%以上的准确率，而SGD需要更多epoch才能接近。Adam的自适应学习率机制能够为不同参数提供个性化的学习率，加速了收敛过程。

**Q2：学习率对训练稳定性有什么影响？**

答：学习率过大（如0.01）会导致训练不稳定，Loss波动剧烈；学习率过小（如0.0001）会导致收敛速度太慢。设置为0.001时，训练过程平稳，Loss持续下降，最终能够收敛到较优解。

**Q3：Batch Size 对模型泛化能力有什么影响？**

答：较大的Batch Size（如256）能够提供更稳定的梯度估计，训练过程更平稳；较小的Batch Size（如64）能带来更好的泛化能力但训练更慢。实验中256的batch size在速度和稳定性之间取得了较好平衡。

**Q4：Early Stopping 是否有效防止了过拟合？**

答：是的，Early Stopping通过监控验证集性能，在模型开始过拟合前停止训练，有效防止了验证集准确率下降。本实验采用保存最佳模型的方式，在整个训练过程中始终保留验证集表现最好的模型。

**Q5：数据增强是否提升了模型的泛化能力？为什么？**

答：数据增强显著提升了模型的泛化能力。通过水平翻转和随机平移，数据集得到了扩充，模型接触到了更多样的数字变体，从而学习到了更鲁棒的特征表示。训练集准确率和验证集准确率的差距缩小，说明泛化能力确实得到了提升。
